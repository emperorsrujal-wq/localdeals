# LocalDeals
Local Business Advertising Platform - Hyperlocal deal discovery and promotion

## Project Structure
```
localdeals/
├── apps/                 # Mobile & Web applications
│   ├── business-app-ios/      # Business app for iOS (Swift/SwiftUI)
│   ├── business-app-android/  # Business app for Android (Kotlin)
│   ├── consumer-app-ios/      # Consumer app for iOS (Swift/SwiftUI)
│   ├── consumer-app-android/  # Consumer app for Android (Kotlin)
│   └── admin-dashboard/       # Admin dashboard (Next.js/React)
├── backend/              # Node.js/NestJS backend API
├── docs/                 # Documentation & PRD
└── scripts/              # Deployment & utility scripts
```

## Apps Overview

### Business Owner App (iOS & Android)
- Create professional flyers in under 2 minutes
- 50+ customizable templates
- Photo capture & editing with AI background removal
- Manage active/expired ads with analytics
- Subscription tiers: Free, Starter, Pro, Enterprise

### Consumer App (iOS & Android)
- Discover nearby deals on interactive map
- Browse weekly ads by category
- Save favorites & follow businesses
- Area scanner for exploring new locations
- Deal redemption & coupon management

### Admin Dashboard (Web)
- Business approval & content moderation
- Revenue & subscription management
- Analytics & reporting
- User & business management
- Template & category configuration

## Tech Stack
- **iOS**: Swift 5.9+, SwiftUI, MVVM, MapKit
- **Android**: Kotlin 1.9+, Jetpack Compose, MVVM, Google Maps SDK
- **Web**: React/Next.js, TypeScript
- **Backend**: Node.js (NestJS) or Python (FastAPI)
- **Database**: PostgreSQL + PostGIS
- **Cache**: Redis
- **Storage**: AWS S3 + CloudFront
- **Maps**: MapKit (iOS), Google Maps (Android/Web)
- **Push**: Firebase Cloud Messaging
